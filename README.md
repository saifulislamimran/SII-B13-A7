# 🤝 KeenKeeper

**KeenKeeper** is a personal relationship management dashboard designed to help you browse, tend, and nurture the connections that matter most. It provides a beautiful interface to track interactions, set relationship goals, and view analytics of your social life.

## ✨ Key Features

* **📱 Fully Responsive Design:** Pixel-perfect layout that looks great on mobile, tablet, and desktop screens.
* **👥 Dynamic Friends Dashboard:** View all your connections in a beautiful grid. Includes real-time **Searching** by name and **Sorting** by name or status.
* **🔍 Smart Filtering & Sorting:** Effortlessly search through friends or timeline history. Sort timeline entries by 'Newest' or 'Oldest'.
* **📊 Analytics & Stats:** Beautiful, pure-CSS Donut/Pie charts visualizing your relationship health (On-track, Almost Due, Overdue) and interaction history.
* **⏱️ Interactive Timeline:** Keeps a track of all your recent interactions (Call, Text, Video, Meetup) with a custom dropdown filter.
* **🖼️ Dynamic Image Loading:** Automatically loads friend avatars from JSON data, with a smart fallback to a default placeholder if the image is missing or broken.
* **🚨 Custom 404 Error Page:** A visually appealing, animated 404 "Page Not Found" screen to handle broken routes gracefully.
* **🔔 Interactive Toasts:** Real-time toast notifications for user actions (Check-ins, Snooze, Archive, Delete).

---

## 🛠️ Technologies Used

* **Frontend Framework:** React.js (Bootstrapped with Vite)
* **Styling:** Tailwind CSS (for fast, utility-first custom designs)
* **Routing:** React Router DOM
* **Icons:** Lucide React
* **Notifications:** React Toastify
* **Deployment:** Vercel & Cloudflare Pages
